# heathcare_system
This is a simple health care management system that works with database to store data.

 #technologies & tools used:
 1. java
 2. javafx (for graphical user interface) also added jfoenix library
 3. wamp server
 4. mysql database
 5. eclipse oxygen
 6. JDK 9
